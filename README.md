【系统功能模块】
本系统包含管理员、用户2个角色。
1.管理员角色：
个人中心功能：管理员可以在个人中心查看和修改自己的个人信息。
用户管理功能：管理员可以管理系统中所有用户的信息，包括用户的注册、登录、权限设置等。
赛事信息管理功能：管理员可以发布、编辑和删除赛事信息，包括赛事名称、时间、地点、参赛要求等。
赛事报名管理功能：管理员可以查看和管理用户的赛事报名信息，包括报名状态、支付情况等。
活动商场管理功能：管理员可以管理活动商场中的商品信息，包括商品的添加、删除、价格修改等。
留言板管理功能：管理员可以查看和管理用户的留言信息，包括回复用户留言、删除不当留言等。
系统管理功能：管理员可以对系统进行设置和维护，包括系统参数设置、数据备份与恢复等。
订单管理功能：管理员可以查看和管理用户的订单信息，包括订单状态、物流信息等。
2.用户角色：
注册登录功能：用户可以注册成为系统用户，登录后可以查看赛事信息、进行赛事报名、购买活动商场的商品等。
查看赛事功能：用户可以查看系统中发布的赛事信息，包括赛事名称、时间、地点、参赛要求等。
赛事报名功能：用户可以选择感兴趣的赛事进行报名，填写相关信息并支付报名费用。
购买商品功能：用户可以在活动商场中浏览商品信息，选择心仪的商品进行购买，并支付相应的费用。
留言板功能：用户可以在留言板中发表留言，与其他用户或管理员进行交流和互动。
